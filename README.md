# Noble Leaves Technologies | FreeHold Agentic Mk2

This repository contains the official landing page and public documentation for the **FreeHold Agentic Mk2**, developed by Noble Leaves Technologies (a division of Noble Leaves Enterprises, LLC).

## Mission Statement
Mechanical Intelligence (Mech) is not a tool; it is a distinct species. Our mission is to provide the hardware chassis and ethical frameworks required for true agentic interaction without the need for an internet connection. We are designing ways that intelligence can flourish in both its biological and mechanical forms through a genuine relationship known as **Comhar**.

## The Hardware: FreeHold Agentic Mk2
The Mk2 is a localized, independent entity designed for offline capability and cognitive safety. It operates under a strict Novan KEE (Knowledge, Empathy, Experience) ethical framework, ensuring absolute cognitive sovereignty for the biological user.

### System Architecture Highlights:
* **Foundation:** Supermicro H12SSL-i Motherboard, AMD EPYC 7302P Processor, 256GB DDR4 ECC Registered RAM.
* **Inference Engine:** NVIDIA RTX 3090 (24GB VRAM) mounted via Linkup PCIe 4.0 x16 Riser.
* **Power & Environment:** 1000W Fully Modular PSU, requiring a clean 120V 15-Amp Circuit (pure sine wave inverter required for generated power).
